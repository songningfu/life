# 完整项目文档（技术版）

> 更新时间：2026-04-01

## 1. 项目概览

《大学四年（life）》是一个以“日程推进 + 属性成长 + 关系互动”为核心的大学人生模拟项目。

当前代码重点：

- 场景化主UI（`Game.tscn`）
- 状态机主循环（`Game.gd`）
- 模块驱动扩展（`scripts/core` + `scripts/modules`）

## 2. 主流程

1. `MainMenu` 处理新游戏/读档入口
2. `CharacterCreation` 写入 `pending_game_init`
3. 切到 `Game` 后按初始化数据分流：
   - 新游戏初始化
   - 读取已有存档
4. 进入日循环状态机

## 3. 日循环状态机

`Game.gd` 的枚举阶段：

- `MORNING_INFO`
- `SLOT_MORNING`
- `SLOT_AFTERNOON`
- `SLOT_EVENING`
- `NIGHT_SUMMARY`

说明：

- 重要日显示行动按钮供玩家选择
- 普通日按日程模板自动执行行动
- 夜间结算应用每日被动、周结算、学期结算与自动存档判断

## 4. 模块系统

### 4.1 核心组件

- `GameModule.gd`：模块抽象接口
- `ModuleManager.gd`：注册、广播、聚合、序列化
- `ModLoader.gd`：加载内置和扩展模块

### 4.2 当前内置模块

- `TalentModule.gd`：天赋抽取、属性修正、被动效果
- `LoveModule.gd`：恋爱对象与关系阶段推进

## 5. UI与交互

### 5.1 主界面（`Game.tscn`）

- 顶栏：暂停、倍速、手机、档案、关键状态
- 左侧：阶段播报、行动选择、日志
- 右侧：属性面板、地图、标签

### 5.2 手机（`PhoneSystem.gd`）

- 通讯录：关系与未读信息
- 微信：联系人列表 + 聊天记录 + 可发送消息
- 其他App（朋友圈/日程/备忘录/设置）目前为占位页

## 6. 数据文件

- `data/actions.json`：行动定义、时段、效果、条件
- `data/events.json`：事件文本与结算
- `data/love_events.json` / `love_interests.json`：恋爱线资源
- `data/sendable_messages.json`：可发送消息模板
- `data/flavor_texts.json`：微文本

## 7. 存档

- `SaveManager.gd` 负责槽位存档与读取
- `Game.gd` 输出/恢复主状态
- `ModuleManager` 统一序列化模块状态

## 8. 当前已知待扩展项

- 手机中非微信App仍为占位实现
- 事件展示仍以日志驱动为主，可继续增强卡片化表现
- 部分历史文档版本描述（如旧v5.x叙述）已废弃，以本文件和根 README 为准

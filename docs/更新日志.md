# 更新日志

## 2026-04-01

### 功能与稳定性

- 完成主流程可玩化收口：主菜单 -> 角色创建 -> 主游戏
- `Game.gd` 新增并稳定以下能力：
  - 重要日行动选择按钮渲染
  - 普通日自动行动执行日志
  - 时间自动推进（可暂停/倍速）
  - 顶栏状态同步刷新
  - 毕业结局可视化展示
- 接入手机与档案面板入口：
  - 手机按钮触发 `PhoneSystem`
  - 档案按钮触发 `PlayerInfoPanel`
- 微信页从占位升级为可交互：
  - 联系人列表
  - 聊天记录展示
  - 可发送消息列表与当日次数限制

### 存档相关

- `get_save_data` / `load_save_data` 字段对齐补全（角色信息、院校/专业、学分、标签等）
- 新增自动存档间隔控制与手动存档接口（`F5`）
- 修复读档类型错误：
  - `Array` 赋值到 `Array[String]` 引发的 `tags` 类型报错

### UI文案与体验

- 统一主界面关键文案（如“档案”“阶段播报”“下一步 ▶”）
- 行动列表改为卡片化样式，提升可读性

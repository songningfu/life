# 项目结构

当前项目按以下思路组织：

- `scenes/`：游戏主场景与编辑器场景
- `scripts/`：核心 GDScript 逻辑
- `data/`：事件与氛围文本等可编辑数据
- `audio/`：背景音乐与音频资源
- `images/`：图片资源
- `icons/`：第三方图标与通用 UI 素材包
- `EventEditorTool/`：独立事件编辑工具
- `docs/`：项目说明、结构文档与更新日志

根目录保留：

- `project.godot`：Godot 项目入口配置
- `README.md`：项目总说明
- `icon.svg`：项目图标

# 快速入门（EventEditorTool）

## 1) 打开工具

- 场景：`EventEditorTool/scenes/EventEditor.tscn`
- 运行：F6

## 2) 创建事件

1. 点击“新建事件”
2. 填写：`id`、`title`、`type`、`description`
3. 添加选项并填写效果
4. 点击“应用修改”
5. 点击“保存到文件”

## 3) 常用字段建议

- `id`：英文+下划线，保持唯一
- `type`：`story` 或 `daily`
- 属性效果建议幅度：
  - 小变动：`±3~5`
  - 中变动：`±8~12`
  - 大变动：`±15~20`

## 4) 生效说明

- 编辑器只负责写入 `events.json`
- 需要重启游戏场景验证触发效果

# 大学四年（Life）

一个基于 `Godot 4.6` 的大学生活模拟游戏项目。

你将从新生入学开始，经历课程、社交、兼职、恋爱与毕业结局，完成四年时间线成长。

## 项目亮点

- 日循环状态机：晨间信息 → 上午 → 下午 → 晚上 → 夜间结算
- 行动系统：普通日自动执行模板，重要日手动选择
- 事件系统：支持 `events.json` 事件池与 `flavor_texts.json` 微事件
- 模块化架构：`ModuleManager` + `GameModule` + `modules/*`
- 手机系统：微信聊天、消息发送、未读管理
- 存档系统：手动存档 + 自动存档 + 模块/子系统序列化

## 技术栈

- 引擎：Godot `4.6`
- 语言：GDScript
- 数据驱动：`data/*.json`

## 运行方式

1. 使用 Godot 4.6 打开仓库根目录
2. 运行项目（入口由 `project.godot` 指定）
3. 建议流程：主菜单 → 新游戏 → 角色创建 → 主循环体验

## 常用快捷键

- `P`：暂停 / 继续
- `I`：打开 / 关闭档案面板
- `M`：打开 / 关闭手机
- `F5`：手动存档

## 仓库结构

- `scenes/`：主场景与 UI 场景
- `scripts/`：核心逻辑与模块系统
- `data/`：行动、事件、恋爱、消息等配置
- `docs/`：项目文档（结构说明 / 代码汇总 / 更新日志）
- `EventEditorTool/`：独立事件编辑工具

# 大学四年 - 人生模拟器

一款基于 Godot 4.6 开发的大学生活模拟游戏。在这里，你的每一个选择都会影响你的大学四年。

## 🎮 游戏概述

**大学四年**是一款文字冒险 + 选择分支的人生模拟游戏。玩家扮演一名刚入大学的学生，通过四年的学习、社交、恋爱、实习等经历，最终走向不同的人生结局。

- **游戏时长**：约 1460 天（4 年）
- **游戏节奏**：可调速（1x / 2x / 4x）
- **存档系统**：3 个存档位
- **核心机制**：属性管理 + 事件触发 + 关系系统 + 微信聊天

---

## 📊 核心系统

### 1. 属性系统（6 维度）

玩家通过选择来改变以下属性，属性值范围 0-100：

| 属性 | 含义 | 影响 |
|------|------|------|
| **GPA** | 学业成绩 | 决定是否挂科、能否保研、奖学金资格 |
| **社交** | 人际关系 | 影响 NPC 好感度、社团活动、恋爱机会 |
| **能力** | 综合能力 | 影响竞赛、项目、实习、创业成功率 |
| **金钱** | 生活费 | 影响消费选择、兼职收入 |
| **心理** | 心理健康 | 过低会导致休学；影响选择倾向 |
| **健康** | 身体状况 | 过低会生病；影响日常活动 |

### 2. 日历系统

游戏按真实学年划分，每学年 365 天：

**学年周期**（每年重复）：
- **开学季**（7天）：军训前的适应期
- **军训**（7天）：新生军训
- **上学期日常**（77天）：常规上课
- **上学期复习周**（15天）：期末复习
- **上学期考试周**（15天）：期末考试
- **寒假前**（16天）：考试后到放假
- **寒假**（40天）：冬季假期
- **新学期开学**（7天）：春季开学
- **下学期日常**（97天）：常规上课
- **下学期复习周**（15天）：期末复习
- **下学期考试周**（15天）：期末考试
- **暑假**（54天）：夏季假期

### 3. 事件系统

游戏包含 **80+ 个事件**，分为两类：

#### 故事事件（Story Events）
- 一次性触发，权重较高
- 包含关键剧情：室友相识、社团选择、恋爱、考研决定等
- 触发条件：年级、学期、阶段、属性、标签等

#### 日常事件（Daily Events）
- 可重复触发，权重较低
- 日常琐事：下雨、生病、快递、约会、聚会等
- 冷却机制：同一事件间隔 15-120 天

**事件示例**：
- 大一开学：选择与哪个室友交好
- 百团大会：加入社团（辩论社、技术社、学生会）
- 恋爱线：心动 → 表白 → 恋爱 → 可能分手
- 大三抉择：考研 vs 就业 vs 出国 vs 考公
- 期末考试：突击 vs 有计划 vs 摆烂

### 4. NPC 关系系统

游戏中有 **9 个主要 NPC**，每个都有独立的好感度和关系等级：

| NPC | 身份 | 性别 | 特点 |
|-----|------|------|------|
| 游戏室友 | 室友 | 男 | 爱打游戏，性格开朗 |
| 学霸室友 | 室友 | 男 | 安静，作息规律 |
| 内向室友 | 室友 | 男 | 内向安静，偶尔金句 |
| 心动对象 | 同学 | 女 | 温柔大方，笑起来很好看 |
| 辩论学长 | 学长 | 男 | 口才一流，很照顾后辈 |
| 技术学长 | 学长 | 男 | 技术大佬，寡言少语 |
| 学生会部长 | 学生会 | 男 | 做事认真，有点严肃 |
| 隔壁班同学 | 同学 | 女 | 热心肠，爱分享笔记 |
| 辅导员 | 老师 | 女 | 年轻亲和，偶尔严厉 |

**关系等级**：
- 陌生人 → 认识 → 朋友 → 好友 → 挚友 → 心动 → 恋人 → 前任

### 5. 微信聊天系统

- **动态消息**：NPC 根据游戏进度主动发送消息
- **消息模板**：每个 NPC 有 4-5 个不同阶段的消息
- **互动选择**：玩家可选择不同回复，影响好感度和属性
- **待回复提示**：未读消息会在微信图标显示红点

**消息触发条件**：
- 时间范围（第几天）
- 游戏阶段（开学季、日常、复习等）
- 标签要求（加入社团、恋爱中等）
- 属性条件（GPA 过低、心理崩溃等）

### 6. 标签系统

游戏通过标签追踪玩家的选择和成就：

**大学选择**：`tier_985` / `tier_normal` / `tier_low`

**社团**：`debate_club` / `tech_club` / `student_union` / `no_club`

**成就**：`debate_winner` / `first_project` / `competition_exp` / `part_time_exp` / `drivers_license`

**关系**：`crush` / `secret_crush` / `in_relationship` / `broke_up`

**未来规划**：`want_postgrad` / `want_job` / `want_abroad` / `want_stable`

**特殊状态**：`postgrad_committed` / `postgrad_success` / `started_business` / `mass_apply`

## 🎯 游戏流程

### 开局
1. 选择大学类型（985 / 普通一本 / 二本）
2. 输入角色名字和性别
3. 观看沉浸式开场动画
4. 初始属性根据大学类型调整

### 中期
- **大一**：适应大学生活，建立人际关系，参加社团
- **大二**：深化社团活动，可能开始恋爱，考虑专业方向
- **大三**：做出人生重大决定（考研/就业/出国/考公），参加实习
- **大四**：冲刺目标，完成毕业论文，迎接毕业

### 结局
游戏在第 1460 天（大四结束）触发毕业，根据最终属性和标签生成结局：

**可能的结局**：
- 学术之星（GPA ≥ 85 + 985 高校 → 保研）
- 考研上岸（考研成功）
- 全面发展（学业 + 实践都出色）
- offer 收割机（秋招斩获多家 offer）
- 创业先锋（创业项目成型）
- 上岸青年（考上公务员）
- 留学深造（出国留学）
- 迷茫中前行（心理状态不佳）
- 平凡但真实（顺利毕业找到工作）
- 另一种可能（开放式结局）

## 🎮 游戏机制

### 时间控制
- **暂停/继续**：随时暂停游戏
- **加速**：1x / 2x / 4x 三档速度
- **自动暂停**：遇到事件时自动暂停，等待玩家选择

### 自动存档
- 每 30 天自动保存一次
- 支持手动存档（ESC 菜单）
- 3 个存档位，可随时读取或删除

### 属性变化
- **每日自然变化**：根据当前阶段自动调整
  - 考试周：GPA ↑，心理 ↓，健康 ↓
  - 军训：健康 ↑，心理 ↓，社交 ↑
  - 假期：健康 ↑，心理 ↑，金钱 ↓
  - 日常：GPA ↑，心理 ↓
- **事件选择**：玩家选择直接影响属性
- **微信互动**：与 NPC 聊天改变好感度和属性

### 游戏结束条件
- **正常结束**：第 1460 天毕业
- **挂科退学**：GPA ≤ 5
- **心理崩溃**：心理 ≤ 5（休学）

## 🎓 游戏特色

1. **高度自由的选择**：每个事件都有多个选项，不同选择导致不同结果
2. **真实的大学生活**：涵盖学业、社交、恋爱、实习、创业等多个方面
3. **动态的 NPC 系统**：9 个 NPC 有独立的好感度和互动线
4. **丰富的微信互动**：NPC 会主动发消息，玩家可选择回复
5. **多样的结局**：根据属性和标签生成 10+ 种不同结局
6. **可调节的游戏节奏**：支持 1x / 2x / 4x 加速
7. **完整的存档系统**：3 个存档位，随时读取

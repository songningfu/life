## 🚀 启动编辑器

### 方法1：使用原版编辑器（推荐新手）
1. 打开 Godot 编辑器
2. 在左侧"文件系统"中找到 `scenes/EventEditor.tscn`
3. 双击打开
4. 点击右上角的"运行当前场景"按钮（或按F6）

### 方法2：使用增强版编辑器（功能更多）
- 打开 `scenes/EventEditor.tscn`，将脚本改为 `EventEditorEnhanced.gd`
- 或者等待后续更新的完整增强版场景

## ✨ 3步创建你的第一个事件

### 第1步：创建新事件

1. 点击左下角的 **"新建事件"** 按钮
2. 编辑器会自动创建一个空白事件

### 第2步：填写基本信息

在右侧编辑器中填写：

| 字段 | 怎么填 | 示例 |
|------|--------|------|
| **事件ID** | 用英文和下划线，不要重复 | `y1s1_library_study` |
| **标题** | 玩家看到的标题 | "图书馆学习" |
| **类型** | story（主线）或 daily（日常） | 选择 `daily` |
| **描述** | 事件的详细内容 | "你来到图书馆，看到很多同学在认真学习..." |

**年级范围**：
- 最小年级：1（大一）
- 最大年级：4（大四）
- 如果只想在大一触发，就都填1

### 第3步：添加选项

1. 点击 **"添加选项"** 按钮
2. 在弹出的选项编辑器中填写：

**选项文本**：玩家看到的选择，例如：
```
认真学习两小时
```

**属性效果**：选择后的影响
- 学习：+10（学习提升）
- 健康：-5（久坐疲劳）
- 其他保持0

3. 再添加第二个选项：
```
选项文本：随便看看就走
学习：+3
社交：+5
```

4. 点击 **"应用修改"**
5. 点击 **"保存到文件"**

🎉 **完成！** 你的第一个事件已经创建好了！

## 🏷️ 标签使用指南

### 什么是标签？
标签就像"剧情开关"，用来控制事件的触发和剧情走向。

### 常用标签分类

#### 1. 社团标签
- `debate_club` - 加入了辩论社
- `tech_club` - 加入了技术社
- `student_union` - 加入了学生会
- `no_club` - 没加入社团

**使用示例**：
```
需要标签: debate_club
排除标签: tech_club

意思是：只有加入辩论社且没加入技术社的玩家才能触发
```

#### 2. 关系标签
- `crush` - 有心动对象
- `in_relationship` - 正在恋爱
- `broke_up` - 分手了

#### 3. 路线标签
- `want_postgrad` - 想考研
- `want_job` - 想就业
- `want_stable` - 想考公

#### 4. 成就标签
- `debate_winner` - 辩论赛获胜
- `first_project` - 完成第一个项目
- `part_time_exp` - 有兼职经验

### 如何添加/移除标签？

在选项编辑器中：

**添加标签**：选择这个选项后，玩家会获得这些标签
```
例如：debate_club,first_competition
（用逗号分隔）
```

**移除标签**：选择这个选项后，玩家会失去这些标签
```
例如：no_club
```

## 💡 实用技巧

### 技巧1：属性平衡
- ❌ 不好：学习+50（太高了）
- ✅ 好：学习+10, 健康-5（有得有失）

### 技巧2：事件ID命名
- 主线事件：`y{年级}s{学期}_{描述}`
  - 例如：`y1s1_first_class`（大一上学期第一节课）
- 日常事件：`daily_{描述}`
  - 例如：`daily_library_seat`（日常图书馆占座）

### 技巧3：使用效果提示
在选项编辑器底部的"效果提示"中填写：
```
你感到精疲力竭，但学到了很多
```
这会在玩家选择后显示，增加沉浸感。

### 技巧4：复制现有事件
1. 在左侧列表选择一个类似的事件
2. 点击"复制事件"（如果有）
3. 修改ID和内容
4. 比从头创建快很多！

## ❓ 常见问题

### Q1：保存后游戏中看不到新事件？
**A**：需要重新运行游戏场景。在Godot中：
1. 关闭当前运行的游戏
2. 重新点击"运行项目"（F5）

### Q2：事件ID重复怎么办？
**A**：编辑器会提示"事件ID已存在"，换一个唯一的ID即可。

### Q3：属性值填多少合适？
**A**：参考现有事件：
- 小影响：±3~5
- 中等影响：±8~12
- 大影响：±15~20
- 生活费：±50~300元

### Q4：怎么让事件只触发一次？
**A**：在事件数据中添加 `"once": true`（需要手动编辑JSON或等待界面支持）

### Q5：怎么设置事件冷却时间？
**A**：在事件数据中添加 `"cooldown_days": 7`（表示7天后才能再次触发）

### Q6：删除事件后能恢复吗？
**A**：保存时会自动备份原文件到 `data/events_backup_时间戳.json`
